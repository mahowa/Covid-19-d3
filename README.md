# Covid-19-d3
#HACKTOBERFEST

D3 Examples using real life data

NY Times Live Dataset: https://github.com/nytimes/covid-19-data/tree/master/live

    - How to custom style a chart

    - Heatmap of us
7.  Add tool tips
